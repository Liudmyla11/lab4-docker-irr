# Лабораторна робота №4

## Тема
Контейнеризація сервісу (Docker)

## Модель
Пошук IRR методами Ньютона та бісекції (5 семестр)

## Опис
Реалізовано математичну модель для знаходження внутрішньої норми прибутку (IRR). Програма порівнює ефективність ітераційного методу Ньютона–Рафсона та методу бісекції для ординарних і неординарних грошових потоків.

## Вимоги
* Встановлений Docker (або середовище GitHub Codespaces / Gitpod)

## Послідовність запуску через Docker

1. **Збірка Docker-образу:**
`docker build -t lab4-model .`

2. **Запуск базового контейнера:**
`docker run --name lab4-container lab4-model`

3. **Налаштування портів:**
`docker run -p 5000:5000 lab4-model`

4. **Запуск зі змінними середовища (варіант 10 - парний):**
`docker run -e STUDENT_NAME="Гетьманенко Людмила" -e GROUP="АІ-231" -e MODE="eco" lab4-model`

5. **Налаштування Docker network:**

Створення мережі:
`docker network create lab-net`

Запуск контейнера в створеній мережі:
`docker run --network lab-net lab4-model`
