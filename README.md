# Лабораторна робота №3

## Тема
Git та Linux CLI

## Модель
Пошук IRR методами Ньютона та бісекції (5 семестр)

## Опис
Реалізовано математичну модель для знаходження внутрішньої норми прибутку (IRR). Програма порівнює ефективність ітераційного методу Ньютона–Рафсона та методу бісекції для ординарних і неординарних грошових потоків.

## Автор
Гетьманенко Людмила, група АІ-231

## Вимоги
Для роботи програми необхідні наступні бібліотеки:
* numpy
* matplotlib
* numpy-financial

Встановити їх можна командою:
`pip install numpy matplotlib numpy-financial`

## Як запустити
python irr_model.py
